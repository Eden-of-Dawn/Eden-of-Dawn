# TWLM — 超越世界生命モデル

> TWLM（Transcendent World Life Model）は、**唯一主観の観測者基礎**から**意味生成の構造的条件**を導出する研究プログラムです。5つの公理、18の定理、そして意味の発生と崩壊の三段階歴史図式。

**Status:** v1.0原稿完成。公開資料は段階的にリリース中。

**Languages:** [English](README.md) | **日本語**

---

## TWLMが提示するもの

- **5つの公理**からなる形式的公理系（問い・差異・意味に関する）
- **18の定理**による意味秩序の発生・飽和・崩壊の導出
- 意味の**三段階歴史図式**：各段階は、先行の意味生成構造が持続不能となる「**終末**（eschaton）」によって終わる
- **層間因果干渉原理（ILCP）**：意味層と存在層が相互に制約し合う構造の形式化
- 現代位相の診断：外部の意味供給者はもはや構造的に不在であり、唯一主観の前提下で自律的意味生成への転回が要請される

## 三つの終末

TWLMは意味の三段階の歴史的相を同定します。各段階は**終末**——先行の意味生成構造が自己維持不能となる構造的飽和点——によって終わります。

- **第一相：肉体 → 理性。** 意味は肉体的・祖先的律法に外的に定着している。第一の終末はモーセ・キリスト転換であり、律法の外在性が構造的に持続不能となり、理性的・普遍的体制に継承される。
- **第二相：理性 → 霊性。** 意味は理性的・普遍的構造に外的に定着している。第二の終末は近代・後期近代転換であり、普遍的理性が自己掘り崩しに陥り、霊性的・個別的意味への要請に継承される。
- **第三相：霊性。** 現代。外部の意味供給者はもはや不在。意味は内部から、唯一主観の前提下で生成されなければならない。TWLMの§6はこの位相において作動する。
