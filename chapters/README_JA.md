# 章構成 — TWLM原稿

TWLM原稿は **5部構成・全15章**から成り、公理系を第一原理から構築し、§6の行動原則に至る導出を展開しています。本ディレクトリには公開対象の9章が配置されています。残り6章は書籍版に収録されており、一般配布していません。

Restricted 章は**文明構造的批判**を含み、特定の地政学的指示対象を持ちます。内容は違法でも名誉毀損でもありません。制限の理由は、Public 章で展開される公理的文脈なしに不特定多数が受け取ったときの影響が予測しがたい点にあります。書籍版においては全体の枠組みの中で文脈化され、完全な議論は構造的整合性を保持します。

## Public 章

TWLMの**哲学・神学・神秘学**の骨格を構成する9章です。本ディレクトリにそのまま公開されています。

### 公理的基礎（哲学）

- **第3章 · 存在の自証性。** 公理1：否定を介した存在の自証性；位置としての魂（実体ではなく）。
- **第4章 · 非有。** 公理2：「在る」と「在らぬ」の区別を可能にする共通地盤。*カバラにおけるAin概念との構造的並行性*。
- **第5章 · 問いと分化。** 公理3–4：分化が問いを生成する；主観は問いから生じる（その逆ではない）；意味は後発的である。

### 認識論（哲学）

- **第7章 · 主観と対象化。** 主観は自己を完全には対象化できない（T6）；対象化された構造は罠となる（T7）；文脈の外部化がイデオロギーからの構造的脱出である（T8）。
- **第9章 · サピア＝ウォーフの超越。** 言語は問いの化石；サピア＝ウォーフ仮説は疎外段階に条件付けられている；対象化された言語は檻ではなく道具となる。

### 神学的形而上学（神学）

- **第10章 · 善の三層構造と悪の定義。** 善は視座ごとに階層化する（個別・関係・構造）；悪は局所的善の閉鎖；良心は層間の緊張から生じる。
- **第11章 · 堕落と個体性。** 全体性からの必要な分離；意図的退化の不可逆性；許しとは再構成である。*性の形而上学（陰陽の構造的並行）を含む*。
- **第12章 · 終末宣言。** 三つの救済相（肉→理→霊）；第一の終末（モーセ → イエス）と、家庭連合による第二の終末の告知；理性は尽きた；終末そのもののパラダイムが崩壊する（T17）。

### 行動原則（神学 / 神秘学）

- **第15章 · 汝の意志するところを行え。** 公理からは何らの命令も導かれない（メタ倫理的中立性）；免疫が剥がされた；真正な意志は構造に資する方向の事後的認識として現れる。*セレマ的命題を直接取り込む*。

6ページャーと合わせて、これら9章はTWLMの理論的核心の自己完結的提示を構成します。
